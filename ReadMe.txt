This repository contains sources and schematics for Samsung NX camera remote shutter and rc interface
For more information see: http://wp.me/p12J9M-41

Files in this repository:
shutter/ - Source files, compile with arduino
shutter.sch - Schematics for camera cable
rx_switch.sch - Schematics for rc interface

The pin definitions are for AtTiny2313, change them if you use different type of
controller
